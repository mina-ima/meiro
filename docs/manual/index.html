1. MEIROってどんなゲーム？

  • オーナー（あなた）の目的: 制限時間が切れるまで、プレイヤーが「規定ポイント」に到達するのを阻止すること。時間切れになればあなたの勝ちです。
  • プレイヤーの目的: 迷路の中のポイント玉を集めて規定ポイントに到達すること。到達した瞬間プレイヤーの勝ちです。

規定ポイントは、あなたが置いた玉の合計点から自動計算されます。どこに何点の玉を置くか、そして罠と予測地点をどう使うかが、あなたの腕の見せどころです。

2. 始め方（ルームの作成）

  1. ゲームのページを開きます。ログインは不要です。
  2. ニックネーム を入力します（2〜10文字、英数字・ハイフン・アンダースコアのみ）。
  3. 「新しいルームを作成」を押すと、6桁のルームコード（英大文字＋数字）が発行されます。
  4. このコードを対戦相手（プレイヤー）に伝えます。

  1. プレイヤーが入室すると「ゲーム開始」ボタンが押せるようになります。
  2. 開始前に 迷路サイズ を選べます: 20×20 または 40×40（初期値は40×40）。迷路はゲーム開始時に自動生成されます。

4. 準備フェーズ — 仕掛けの配置（60秒）

準備フェーズは時間で3段階に分かれ、自動的に切り替わります。60秒しかないので、どこに何を置くかを事前にイメージしておきましょう。

4-1. ポイント玉の配置（最初の40秒）

画面下部の設置ツールで 1点／3点／5点 のボタンを選び、俯瞰マップのマスをクリックして置きます。

迷路サイズ 合計ポイントの下限 配置できる個数
20×20 40点以上 最大12個
40×40 60点以上 最大18個
  • 規定ポイント（プレイヤーの目標）= あなたが置いた合計ポイントの 65%（切り上げ）。例: 合計20点 → 規定13点。
  • 合計が下限に届かないまま時間切れになると、不足分がプレイヤーの初期ポイントとして自動付与されます（規定ポイント−1が上限）。置かなすぎると自分が不利になるので、下限までしっかり置きましょう。
  • 設置ツールには「残り個数／合計点／下限までの不足分」が常に表示されます。

コツ: 高得点の5点玉をゴールから遠い行き止まりに置けば、プレイヤーの移動距離を稼げます。逆にスタート近くに置くと、すぐに取られてしまいます。

4-2. 罠の設置（次の5秒）

  • 罠ツールが自動で選択されます。マップのマスをクリックして設置します。
  • 罠を置ける権利は 最初は1個 です。予測地点の報酬（→4-3）などで増えることがありますが、同時に置いておけるのは 最大2個 までです。
  • 罠は プレイヤーには見えません。踏むと一定時間移動が遅くなり、踏まれた罠は消えます。
  • 置けない場所: スタート付近・ゴール付近・ポイント玉と同じマス。

4-3. 予測地点のマーキング（最後の15秒）

  • 予測ツールが自動で選択されます。「プレイヤーがここを通りそうだ」と思うマスを 最大3箇所 クリックしてマークします。
  • プレイヤーがそのマスを通ると、あなたにランダム報酬（追加の妨害リソース）が入ります。
  • プレイヤーには「予測地点を通過！」と通知だけが出ます。何が起きたかは相手には分かりません。

配置操作の共通ルール

  • 配置操作には 約1秒のクールダウン があり、連打はできません。
  • プレイヤーのいるマスから縦・横の距離の合計が2マス以内（斜めは数えません）は編集禁止エリア です。

5. 探索フェーズ中のオーナー画面

探索が始まると、俯瞰マップでプレイヤーの動きをリアルタイムに観察できます。

俯瞰マップの操作

ボタン 動作
－／＋ ズームアウト／ズームイン
↑←→↓ マップを上下左右にスクロール（パン）

マップに表示されるもの

  • プレイヤーの現在位置と向き（青い三角マーカー。リアルタイムに動きます）
  • 配置済みの ポイント玉（点数の数字つきマーカー。取られると消えます）
  • 罠 と 予測地点（プレイヤーには見えませんが、あなたには見えます）
  • ゴール（黄色の枠）と スタート（青の枠）

情報パネル

罠権利／罠設置数／禁止エリア距離／編集クールダウン／予測地点の残数／ポイント配置数と合計点／残り時間に加えて、プレイヤーの位置・向き・周囲の通路状況 が数値でも確認できます。

6. 勝敗とスコア

  • 規定ポイント = 配置した合計ポイントの65%（切り上げ）。
  • 制限時間切れまでプレイヤーが規定ポイントに届かなければ あなたの勝ち。
  • プレイヤーが規定ポイント以上を獲得した 瞬間にプレイヤーの勝ち（即終了）。
  • ゴールボーナス = 規定ポイントの2割（切り上げ）。プレイヤーがゴールに到達すると加算されますが、ゴール到達だけではゲームは終わりません。

リザルト画面では最終状態の俯瞰マップと、取られた／残ったポイントを確認できます。

7. コラム：プレイヤーは何をしている？

準備フェーズの60秒間、プレイヤーの画面には迷路のさまざまな場所のプレビュー映像クリップが流れています（ゴール地点が必ず1回は映ります）。探索が始まると、プレイヤーはキーボードか画面ボタンで1マスずつ移動し、前方およそ4マスしか見えない暗い迷路でポイント玉を探します。罠も予測地点もミニマップも見えていません。あなたの俯瞰マップがどれだけ有利な情報源か、想像しながら仕掛けを考えましょう。

4. 準備フェーズ中のプレイヤー画面（60秒）

  • この間、あなたはまだ移動できません。オーナーが迷路に仕掛けを置いている時間です。

5. 探索フェーズ — 操作方法

移動は 1マス単位のステップ式 です。キーボードでも画面ボタンでも操作できます。

キー 画面ボタン 動作
W または ↑ ↑ 1マス前進（向きはそのまま）
S または ↓ ↓ 1マス後退（向きはそのまま。後ろ向きには変わりません）
A または ← ← 左を向いて1マス進む（向きも左に変わります）
D または → → 右を向いて1マス進む（向きも右に変わります）
  • その場で向きだけを変える操作はありません。「曲がる」＝「向きを変えて1マス進む」です。
  • キーやボタンを押しっぱなしにすると、1マスずつ止まらずに連続で進めます（壁にぶつかる手前で自動停止します）。
  • 移動できる方向のボタンだけが点灯します。壁の方向のボタンは自動的に暗くなり、押せません。
  • 画面右上に「向き／前後左右が通路か壁か」の案内が常に表示されます。行き止まりでも、この表示を見ればどちらに曲がれるかすぐ分かります。

6. 画面の見かた（3Dビュー）

  • 迷路の中は一人称視点の3D映像で表示されます。視界は前方およそ4マスで、それより先は闇に沈んで見えません。
  • ミニマップはありません。来た道を覚えながら進みましょう。

ポイント玉

ポイント玉は通路に浮かんで光っており、上に「1点」「3点」「5点」のラベルが表示されます。

玉のあるマスに入ると自動で獲得できます。取った瞬間に画面下の「現在ポイント」が増えます。

ゴール

  • ゴールは シアン（青緑）に光る縦の光柱 です。遠くからでも壁ごしの淡い光で気づけることがあります。
  • ゴールに到達すると ゴールボーナス（規定ポイントの2割・切り上げ）が加算されます。
  • ただし ゴールに着いただけではゲームは終わりません。規定ポイントに達していなければ探索は続行です。

  • 罠: プレイヤーには見えません。踏むと一定時間、移動が遅くなります（踏まれた罠は消えます）。
  • 予測地点: これも見えません。通過すると「予測地点を通過！」と通知が出ますが、何が起きたかは教えてもらえません（実はオーナーに追加の妨害リソースが入っています）。

7. ポイントと勝利条件

  • 規定ポイント = オーナーが配置した合計ポイントの 65%（切り上げ）。例: 合計20点 → 規定13点。
  • 現在ポイントが規定ポイント以上になった 瞬間にあなたの勝ち（ゲームは即終了します）。
  • 制限時間切れまで届かなければ オーナーの勝ち です。
  • オーナーの玉の置き方が少なすぎた場合、不足分が 初期ポイント補填 としてあなたに最初から付与されます（HUDに表示されます）。

リザルト画面には最終スコアと規定ポイントが表示されます。

8. コラム：オーナーは何をしている？

準備フェーズの60秒間、オーナーは迷路を上から見下ろしながら、ポイント玉（1点/3点/5点）の配置 → 罠の設置 → 予測地点のマーキングを順番に行っています。探索が始まると、オーナーの画面にはあなたの位置と向きがリアルタイムで表示されます。ただし探索中のオーナーは見守ることしかできません。勝負はオーナーの「仕掛け」対あなたの「探索力」です。
